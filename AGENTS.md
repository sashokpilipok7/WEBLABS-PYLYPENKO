# AGENTS.md — E-commerce Project (HTML/CSS/JS)

## Огляд проекту
Цей проект є простою e-commerce платформою (frontend-only).
- **Стек**: Vanilla HTML5, CSS3, ES6 JavaScript.
- **Структура**: Односторінковий додаток та набір статичних зовнішніх сторін.

## Настанови для агента
- Використовуй семантичний HTML5 (header, main, section, footer).
- Дотримуйся методології яка вже є файлі для іменування CSS класів.


## SEO та Мета-дані
- **Semantic SEO**: Кожна сторінка повинна мати унікальний `<title>` та `<meta name="description">`.
- **Заголовки**: Суворе дотримання ієрархії `<h1>`-`<h6>`. Лише один `<h1>` на сторінку.
- **Зображення**: Обов'язкові атрибути `alt` для всіх `<img>`. Використовуй `loading="lazy"` для товарів поза першим екраном.
- **Open Graph**: Для карток товарів та головної обов'язково генеруй:
  - `og:title`, `og:description`, `og:image`, `og:type="product"`.

## Технічні вимоги до верстки

### 1. Головна сторінка (Navigation)
- Список категорій товарів має бути реалізований як:
  ```html
  <aside>
    <nav aria-label="Категорії товарів">
      <ul>...</ul>
    </nav>
  </aside>

### 2. Каталог та Фільтрація (Accessibility)
- Для повторюваних елементів (карток, фільтрів) використовуй ARIA-ролі.
- Блок фільтрів: role="search" або role="group" з відповідними aria-labelledby.

### 3. Картка товару (Microdata & Semantics)
- Ціна: Використовуй тег <data> з атрибутом value (наприклад, <data value="450">450 грн</data>).
- Характеристики: Тільки через список описів:
    ```html
    <dl>
        <dt>Колір</dt>
        <dd>Чорний</dd>
    </dl>
